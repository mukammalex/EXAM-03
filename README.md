# Catalyst Analytics

##  About The Project

Catalyst Analytics is a modern and responsive landing page developed using **HTML** and **Tailwind CSS**. The website showcases advanced digital services in artificial intelligence, blockchain development, algorithmic trading, data engineering, predictive analytics, and MLOps solutions.

The project focuses on creating a clean user interface, modern layouts, and responsive design principles to deliver a professional user experience.

##  Features

* Responsive design for different screen sizes
* Modern dark-themed UI
* Hero section with attractive visuals
* Services section with detailed service cards
* Interactive buttons and hover effects
* Professional footer with contact information and social media links
* Built entirely with Tailwind CSS utility classes

##  Technologies Used

* HTML5
* Tailwind CSS
* JavaScript
* Git & GitHub
